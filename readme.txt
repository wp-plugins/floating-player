== Description ==

Floating Player Allows You to add an introduction video to your website.If you wan't to promote any introductory video related to your site you can add it via plug in 

Features include:

*	Add Introduction Videos
*	Manage Position of player on screen
*	Add MP4/FLV
*	Set Height and Width of player
*	Transparent Player Background



== Installation ==

1. Download and unzip the file.
2. Log in to WordPress and go to Plugins 
3. Upload the Video in My Player Option
4. Click on the Activate link for Custom Video Display.
